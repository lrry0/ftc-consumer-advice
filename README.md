# Federal Trade Commission | Protecting America's Consumers

This is an interactive portal replica of the Federal Trade Commission (FTC) Consumer Advice website, featuring multi-page client-side routing, scams tracking, and compliance record registries.

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `pnpm install`
2. Set the `GEMINI_API_KEY` in `.env.local` to your Gemini API key
3. Run the app:
   `pnpm run dev`
